## Movie api review
This is a simple REST api created using node.js and express.js which manages movie reviews.

## features
Add a new Movie reviw(POST)
Get all movie reviews(GET)
Get a single movie review by ID(GET)
Update an existing movie review(PUT)
Delete a movie review(DELETE)

## Api endpoints
POST/api/movies
GET/api/movies
GET/api/movies/:id
POST/api/movies/:id
DELET/api/movies/:id
GET/api/movies/:id

## Example POST Body
{
  "movieTitle": "Inception",
  "director": "Christopher Nolan",
  "reviewText": "Amazing movie!",
  "rating": 9
}

## How to run
npm install
npm start

It runs on:  http://localhost:3000
